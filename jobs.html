Join Our Development Team

Become a developer or helper for Spargat Group. Help us create and evolve Blockman GO solutions.

Project Information

This is a closed-source project, and the source code will never be opened to the public. Only trusted volunteers and developers will have access to certain parts of the codebase.

Privileges

  • Access to the project for creating public/private servers of CubicalCore, BlockmanLORD, and Blockman Launcher.
  • Use parts of the code for any developer-related projects.
  • Database access for team projects.

Rules

  • No leaking the project's sources to anyone.
  • Active contributions are required (developers must commit regularly, especially within a month).
  • Knowledge of languages like C, Java, Kotlin, C++, Bash, and Git is essential.
  • When making servers or using the code, copyright is required, and credit must be given for usage of the project.

Skills Required

  • Proficiency in C, Java, Kotlin, C++, Bash, and Git.
  • Experience with closed-source game development and server management.
  • Ability to collaborate with a team and make consistent contributions.

How to Apply

If you're interested in joining our team, contact one of the owners for further details and next steps.
